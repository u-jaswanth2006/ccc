Let’s do the S3 lab step by step exactly.

This lab has 3 tasks from your PDF

TASK 1: Create S3 bucket + Upload file + Public access
Step 1: Open S3

Go:

AWS Console → Services → S3

Click:

Create bucket
Step 2: Create bucket

Fill:

Bucket name
(make globally unique)

Example:

jaswanth-public-bucket-2026

Region:
Choose:

us-east-1
Step 3: Public access settings

Very important.

Under:

Block Public Access settings

Uncheck:

Block all public access

AWS warning appears.

Check:

I acknowledge that the bucket will become public

Click:

Create bucket
Step 4: Upload object

Open your bucket.

Go:

Objects tab

Click:

Upload

Choose any file:
Example:

image.jpg

Click:

Upload
Step 5: Make object public

Open uploaded object.

Go:

Permissions

Scroll:

Access Control List (ACL)

Click:

Edit

Under:

Everyone (public access)

Enable:

Read

Save.

Step 6: Test

Go:

Object overview

Copy:

Object URL

Open browser incognito.

If it opens:
SUCCESS.

TASK 2: Versioning + Cross Region Replication
PART A: Versioning
Step 1

Open source bucket:

jaswanth-public-bucket-2026

Go:

Properties

Scroll:

Bucket Versioning

Click:

Edit

Select:

Enable

Save.

Step 2: Test versioning

Upload:

report.pdf

Upload again same filename:

report.pdf

AWS creates new version.

Check:

Objects → Show versions

You’ll see:

version 1
version 2
PART B: Cross Region Replication (CRR)
Step 3: Create destination bucket

Go:

S3 → Create bucket

Name:

jaswanth-crr-destination-2026

Region:
Choose DIFFERENT region.

Example:

ap-south-1

Create.

Step 4: Enable versioning on destination

Open destination bucket.

Go:

Properties

Enable:

Bucket Versioning
Step 5: Create replication rule

Open SOURCE bucket.

Go:

Management

Open:

Replication rules

Click:

Create replication rule
Step 6: Rule config

Rule name:

my-crr-rule

Status:

Enabled

Scope:

Apply to all objects
Step 7: IAM role

Choose:

Use existing role

Select:

LabRole

(as your PDF says)

Step 8: Destination

Choose:

Another bucket

Select:

jaswanth-crr-destination-2026

Replication:

New objects only

Save.

Step 9: Test replication

Upload new file to SOURCE:

test.png

Wait 1–2 mins.

Open destination bucket.

You should see:

test.png

SUCCESS.
